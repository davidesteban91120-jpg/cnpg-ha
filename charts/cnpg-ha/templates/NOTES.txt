cnpg-ha {{ .Chart.AppVersion }} installed in namespace `{{ .Release.Namespace }}` as release `{{ .Release.Name }}`.

1. Check that the controller is up:

   kubectl -n {{ .Release.Namespace }} rollout status deploy/{{ include "cnpg-ha.fullname" . }}-controller-manager

2. Tail logs (current level: {{ .Values.log.level }}):

   kubectl -n {{ .Release.Namespace }} logs -l app.kubernetes.io/instance={{ .Release.Name }} -f

   kubectl get hacluster -A

   helm upgrade {{ .Release.Name }} {{ .Chart.Name }} -n {{ .Release.Namespace }} --reuse-values --set log.level=debug
